# Day 01｜自我介绍 + AGV系统总览 / 机场值机与安检

日期：2026-05-18  
时长：45分钟  
技术主题：Self-introduction and AGV system overview  
生活出行：Airport check-in and security

## 45分钟安排
- 0–25分钟：技术现场英语：读句子 → 跟读 → 看中文说英文。
- 25–35分钟：生活出行情景：读短句 → 模拟问答。
- 35–42分钟：发音：重点词慢读3遍，再放进句子。
- 42–45分钟：复盘：写下今天最卡的3个词/句。

## 必背5句
1. I am here to support the deployment and testing of our AGV control system.  
   我这次来是支持AGV中控系统的部署和测试。
2. Our system manages tasks, traffic, charging, and real-time vehicle status.  
   我们的系统管理任务、交通、充电和车辆实时状态。
3. Could you please speak a little slower? I want to make sure I understand correctly.  
   您能说慢一点吗？我想确保我理解正确。
4. I have one checked bag and one carry-on bag.  
   我有一件托运行李和一件随身行李。
5. Is this the line for security check?  
   这是安检队伍吗？

## 词汇与例句
| 词汇 | 中文 | 例句 |
|---|---|---|
| deployment | 部署 | The deployment will start after the network is ready. |
| commissioning | 调试/投运 | We need two days for commissioning. |
| fleet | 车队 | The fleet has twenty AGVs. |
| traffic control | 交通管制/调度 | Traffic control prevents deadlocks. |
| task assignment | 任务分配 | Task assignment is handled by the central system. |
| real-time status | 实时状态 | We monitor real-time status on the dashboard. |
| checked bag | 托运行李 | I have one checked bag. |
| boarding pass | 登机牌 | Could you print my boarding pass? |
| security check | 安检 | Where is the security check? |

## 技术口语脚本 / 微对话
- **Customer:** Could you briefly introduce your role?
- **You:** Sure. I am here to support deployment, testing, and troubleshooting of the AGV control system.
- **Customer:** What does the system mainly do?
- **You:** It manages orders, vehicle status, traffic rules, charging, and monitoring. If any issue happens, we will check logs and reproduce it together.

## 技术口语脚本 / 微对话
- **You:** Could you please repeat the question?
- **Customer:** Sure. We want to know whether the system can handle ten AGVs at the same time.
- **You:** Thanks. Yes, the system is designed for fleet management. But we should confirm the network, server resources, traffic rules, and test results.
- **Customer:** Can you summarize today’s progress?
- **You:** Yes. Today we finished deployment verification, checked network ports, tested basic orders, and listed tomorrow’s action items.
