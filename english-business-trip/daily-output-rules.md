# 每日 Markdown + HTML 输出规则

更新时间：2026-05-12

## 用户要求

每天训练材料同时输出：

1. Markdown：方便长期保存、复盘、修改。
2. HTML：方便浏览器打开和打印，用户可以拿着练习。

## 输出路径

每日材料放在：

`/home/hukai/personalGood/04-knowledge-base/english-business-trip/daily/`

命名规则：

- Markdown：`day-XX-topic.md`
- HTML：`day-XX-topic.html`

示例：

- `day-01-agv-airport.html`
- `day-01-agv-airport.md`

## 每日内容结构

每天 45 分钟，固定 3 块：

1. 技术主线：AGV / VDA5050 / Linux 内网部署。
2. 生活主线：机场、飞机、入境、打车、酒店、餐厅、购物、紧急情况。
3. 发音主线：当天 5-8 个重点词，拆音节、重音、跟读句。

推荐比例：

- 25 分钟技术现场英语
- 10 分钟生活场景英语
- 7 分钟发音
- 3 分钟复盘

## HTML 打印要求

1. A4 打印友好。
2. 中英对照。
3. 英文句子加粗，中文解释在下方。
4. 每天不超过 3 页为佳。
5. 保留“今日必须背熟 5 句”。
6. 保留“今日发音重点”。
7. 保留“复盘区”。
