# Day 01：AGV 自我介绍 + 机场值机/安检

日期：2026-05-13
总时长：45 分钟
主题：AGV 现场开场 + 机场值机/安检

## 今日目标

1. 能说出自己负责 AGV 中控系统部署和技术支持。
2. 能说出系统部署在客户本地 Linux 内网服务器上。
3. 能在机场办理值机、询问座位、通过安检时听不懂会求助。
4. 重点练 8 个高频词的发音。

## 0-5 分钟：热身保命句

每句慢速读 2 遍，正常速度读 2 遍。

1. Sorry, could you please repeat that more slowly?
   不好意思，可以请你说慢一点再重复一遍吗？

2. Could you please type it in the chat?
   可以麻烦你把它打在聊天窗口里吗？

3. Let me confirm my understanding.
   我确认一下我的理解。

4. I need a few minutes to verify this.
   我需要几分钟确认一下。

5. I will confirm it in writing after the meeting.
   我会在会后用文字确认。

## 5-18 分钟：AGV 现场自我介绍

先看稿读 3 遍，再半脱稿说 2 遍。

Hi, my name is XXX.

I’m responsible for the deployment and technical support for this project.

This is an AGV central control system for warehouse and logistics operations.

The system will run on the local Linux server in the internal network.

Today, we will check the environment first, then deploy the system step by step.

My spoken English is not perfect, so I may confirm key points in writing to avoid misunderstanding.

Thank you.

## 今日必须背熟 5 句

1. I’m responsible for the deployment and technical support.
   我负责部署和技术支持。

2. This is an AGV central control system.
   这是一个 AGV 中控系统。

3. It is used for warehouse and logistics operations.
   它用于仓储和物流作业。

4. The system will run on the local Linux server.
   系统会运行在本地 Linux 服务器上。

5. The server is in the internal network.
   服务器在内网中。

## 28-38 分钟：生活场景：机场值机 + 安检

### 值机 Check-in

I’d like to check in for my flight.
我想办理登机手续。

Here is my passport.
这是我的护照。

Can I have an aisle seat?
可以给我一个靠过道座位吗？

Can I have a window seat?
可以给我一个靠窗座位吗？

How many bags can I check in?
我可以托运几件行李？

Where is the security check?
安检在哪里？

### 安检 Security

Do I need to take off my shoes?
我需要脱鞋吗？

Do I need to take out my laptop?
我需要拿出笔记本电脑吗？

This is my laptop.
这是我的笔记本电脑。

This is my power bank.
这是我的充电宝。

I don’t understand. Could you please repeat that more slowly?
我没听懂。可以请你说慢一点再重复一遍吗？

## 38-42 分钟：今日发音重点

1. deployment：de-PLOY-ment，重音在 PLOY
2. central：CEN-tral，重音在 CEN
3. control：con-TROL，重音在 TROL
4. warehouse：WARE-house，重音在 WARE
5. logistics：lo-GIS-tics，重音在 GIS
6. local：LO-cal，重音在 LO
7. internal：in-TER-nal，重音在 TER
8. network：NET-work，重音在 NET

跟读节奏：

This is an AGV / central control system.

The system will run / on the local Linux server / in the internal network.
