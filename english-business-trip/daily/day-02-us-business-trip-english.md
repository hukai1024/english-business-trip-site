## 技术口语脚本 / 微对话
- **You:** Before installation, I need to check the server environment.
- **Customer:** What do you need?
- **You:** Linux version, CPU, memory, disk space, IP address, firewall rules, and permissions for the deployment directory.
- **Customer:** Can you install it now?
- **You:** Yes, after the environment check is complete, I can start the installation and then verify the service status.
